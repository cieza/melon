Para executar é necessário:

1 - Baixar NodeJS
https://nodejs.org/en/

2 - Git clone projeto
git clone https://github.com/cieza/melon

3 - Entrar na pasta do projeto e executar:
npm install

4 - Para usar o index.js:
node index.js
//Este script faz o download dos scans da pasta informada

5 - Para usar o createScan.js:
node createScan.js <arquivo_com_alvos>
//Este script irá criar um scan para cada linha do arquivo
